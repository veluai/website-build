# Image SEO Alt Text Standard

## Purpose

This document defines Velu's image SEO and alt text standard for public website images.

Image alt text should support accessibility first and SEO second. Image naming, alt text, and schema image references should reinforce page context, nonprofit specialization, and accurate visual description without keyword stuffing.

## Scope

This standard applies to:

- Homepage images
- Service page images
- Framework graphics
- Service pillar images
- Team images
- Lead magnet images
- Logo images
- Future blog and resource images

This standard governs image description and SEO treatment. It does not replace brand design standards, image production standards, or page layout decisions.

## Alt Text Philosophy

- Alt text should describe the image accurately.
- Alt text should support page context.
- Alt text should include nonprofit-specific terminology when natural.
- Alt text should not keyword stuff.
- Alt text should support accessibility first and SEO second.
- Decorative images may use empty alt text if appropriate.

Alt text should be useful to a person who cannot see the image. Search relevance is valuable only when it follows from accurate description and page context.

## Image Naming Categories

Velu website images should be categorized by role:

- Hero images
- Framework images
- Service pillar images
- Trust-building images
- Lead magnet images
- Logo images
- Review or testimonial images

Image filenames and alt text should be specific enough to help future maintainers understand why the image exists on the page.

## Alt Text Patterns

Hero image:

"Nonprofit accounting and advisory services helping nonprofit leaders gain financial clarity and sustainable growth."

Service pillar image:

"[Service topic] for nonprofits including [primary outcome or capability]."

Framework image:

"Velu financial growth framework showing [relevant outcome pillars]."

Lead magnet image:

"Nonprofit startup guide for founders building a strong financial foundation."

Team/collaboration image:

"Velu team collaboration supporting nonprofit accounting and advisory services."

Logo:

"Velu nonprofit accounting and advisory firm logo."

These patterns are starting points, not mandatory repeated text. Final alt text should be adjusted to match the actual image.

## Current Known Image Use

Current known image categories include:

- Homepage hero image
- Homepage collaboration image
- Homepage lead magnet image
- Outsourced Accounting service pillar images
- Client Advisory service pillar images

Future image inventories should document the page, image role, image URL or asset location, alt text, and verification status when that information is available.

## Schema Relationship

- Alt text is primarily implemented in HighLevel or HTML image settings.
- Schema image URLs should reference actual image URLs.
- `ImageObject` descriptions are optional and lower priority than accurate alt text.

Schema image references should not be treated as a substitute for accessible image alt text. Image URLs used in schema must resolve before a page is marked complete in the page schema inventory.

## Future AI Agent Instructions

- Do not invent image contents.
- Write alt text based on actual image and page context.
- Prefer nonprofit-specific descriptions where accurate.
- Avoid repeating the exact same alt text across multiple images.
- Keep alt text concise and natural.
- Do not keyword stuff image alt text or filenames.
- Distinguish decorative images from meaningful content images.
