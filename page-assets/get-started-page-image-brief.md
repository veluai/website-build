## Visible Page Image Strategy

The Get Started page intentionally uses no visible content images.

The page should remain focused on:

1. The booking introduction
2. The appointment identity and duration
3. The embedded GoHighLevel calendar
4. The fallback email contact

No founder portrait, hero image, illustration, service graphic, icon system, or testimonial imagery should be introduced.

## Rationale

The page has one conversion objective: scheduling a discovery call.

Additional images would compete with the embedded calendar and add unnecessary visual weight. Tyler Wilcox, CPA is identified clearly in the page copy and appointment title, so a visible portrait is not required.

The embedded calendar should remain the dominant visual element.
